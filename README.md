# graspnet_ros

ROS Implementation of [GraspNet](https://githjub.com/graspnet/graspnet-baseline)

## Prerequisites
* Anaconda3
* Ubuntu 20.04
* ROS Noetic

## Installation

Create Python Environment
```sh
conda create -n graspnet_ros python=3.8
```

Install graspnetAPI.
```sh
git clone https://github.com/graspnet/graspnetAPI.git
cd graspnetAPI
pip install .
```
